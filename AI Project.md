GPU Price Forecast AI

แอปพลิเคชันนี้เป็นระบบ พยากรณ์ราคา GPU ในอนาคต โดยใช้เทคนิค AI Time Series Forecasting เพื่อช่วยวิเคราะห์แนวโน้มราคาการ์ดจอในอนาคต

ระบบถูกพัฒนาเป็น Web Application ด้วย Streamlit ทำให้ผู้ใช้สามารถเลือก GPU รุ่นที่ต้องการ และดูกราฟการคาดการณ์ราคาในอนาคตได้

กลุ่มเป้าหมาย
	•	ผู้ที่ต้องการซื้อการ์ดจอ
	•	นักวิเคราะห์ตลาดฮาร์ดแวร์
	•	ผู้ที่สนใจแนวโน้มราคาการ์ดจอ

ปัญหาที่ระบบช่วยแก้
	•	ไม่รู้ว่าราคา GPU จะขึ้นหรือลงในอนาคต
	•	ต้องการข้อมูลช่วยตัดสินใจซื้อ

การสะท้อนคิด
1. สิ่งที่ได้เรียนรู้
ได้เรียนรู้การพัฒนาโมเดลพยากรณ์ราคา GPU โดยใช้ Prophet รวมถึงการเตรียมข้อมูล เช่น การจัดรูปแบบข้อมูลเวลา
การทำ Log Transform และการประเมินผลโมเดลด้วยค่า MAPE

2. สิ่งที่ทำได้ดี
สามารถพัฒนาโค้ดสำหรับฝึกโมเดลและพยากรณ์ข้อมูล GPU หลายรุ่นได้ โค้ดมีโครงสร้างชัดเจน ใช้การวนลูปและการจัดเก็บผลลัพธ์อย่างเป็นระบบ
และสามารถบันทึกผลลัพธ์ออกมาเป็นไฟล์ CSV เพื่อนำไปวิเคราะห์ต่อได้

3. สิ่งที่ต้องปรับปรุงพัฒนาในอนาคต
ในอนาคตควรพัฒนาโมเดลให้มีความแม่นยำมากขึ้น เช่น ทดลองใช้โมเดลอื่นเพิ่มเติม เช่น ARIMA, LSTM หรือ Machine Learning Model
data ที่นำ ต้องมี Feature มากว่านี้
